• LANDING PAGE
    • Slider
      • Contain a 5-10 slides slider that give interesting information about the benefit of opting for a vegetarian meal.
      • Slider will pull this info from a JSON format file.
    • Headline - H1
      • Welcome heading and app name.
    • Supporting Copy - Couple of sentences about the goal of this app
    • Filtering Logic ( EVENT ) for Main Content - Primary Action
      • Set of Selector Options
        • Do you want to cook?
        • Do you want to eat out?
      • Event listener will listen to the click and open the corresponding search box below it .
    • Search Box - Cook option selected
      • For the basic version user is prompted to enter keywords to search for a vegetarian recipe.
      • Keywords could be ingredients, name of the dish or cuisine name .
      • Examples of search "carrots potatoes" , "quick indian curry ", "vegan pad thai" etc..
    • Search Box - Dine out option selected
      • Search box with an input field to obtain a destination city.
      • keywords can be sandiego, bangkok, montreal etc..
    • API Queries
      • Based on the initial selector chosen the endpoint URL is selected.
      • Query string from the search keywords is concatenated with the string "vegetarian" .
      • The URL and query is sent in a AJAX request and corresponding data is obtained either with a single and nested API calls.
    • Search Results Page (Dynamic)
      • The search page will be dynamically added below the landing page and above the info page, with some kind of transition .
      • Headline - H2
      • Listings are Secondary Actions
        • Ul / LI Listings
          • The JSON data returned from AJAX call is parsed and html is generated in lists
          • Ideally show 4-5 options on the page and have a "Load more" button to retrieve 5 more.
          • Recipe Search
            • The list will contain an image of the recipe
            • recipe name
            • calorie details (if applicable)
            • The list item will be linked to take the user to the actual recipe page.
            • Phase 2 : On clicking the image show a pop up modal/lightbox with recipe instructions , ingredients etc and in there provide a url to take user to original recipe website.
          • Restaurant Search
            • The list will contain the restaurant name
            • Image for the restaurant if applicable.
            • Clicking on the image will take user to the restaurant website.
            • Phase 2 : Find a mechanism to gather user zipcode and show distance/directions to the restaurant.
    • Information Page
      • This page will have article blurbs to give more in depth information to inpire folks to eat more vegetarian in a box format.
      • Ideas : comic page style info, web md article, wikipedia article
      • Phase 2 : Show meetups and groups in the user neighborhood that support vegetarianism or veganism.
